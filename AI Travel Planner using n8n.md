#### **# AI Travel Planner using n8n and OpenRouter**

##### 

###### **## Project Overview**

##### 

AI Travel Planner is an intelligent travel assistant developed using n8n and OpenRouter AI models. The system helps users plan trips by generating personalized travel recommendations, budget estimates, hotel suggestions, and travel itineraries based on user preferences.

##### 

###### **## Features**



\* Destination recommendations

\* Travel itinerary generation

\* Budget planning

\* Hotel suggestions

\* AI-powered travel assistance

\* Interactive chat interface



###### **## Technologies Used**

##### 

\* n8n

\* OpenRouter AI

\* AI Agent

\* HTTP Request

##### 

###### \## **Workflow**

##### 

1\. User enters travel destination and preferences.

2\. Chat Trigger receives the request.

3\. AI Agent processes the travel query.

4\. OpenRouter AI generates recommendations and itinerary.

5\. Travel details are returned to the user.

##### 

###### **## Nodes Used**



\* Chat Trigger

\* AI Agent

\* OpenRouter Chat Model

\* Memory

\* Set Node

\* HTTP Request (Optional)

\* Respond to Chat

#####

\* Easy-to-use conversational interface
